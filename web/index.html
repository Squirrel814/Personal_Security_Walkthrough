Stay calm. Document everything. You are in control.

Educational step-by-step self-help for scare-tactic files and targeted harassment. Investigate on your own systems, block ethically, escalate when needed. Not legal advice. Not a substitute for professional incident response.

Action ladder

Every path follows the same safe order:

  1. Document — inventory, screenshots, metadata. Never delete first.
  2. Comprehension gate — pause until you understand what you found.
  3. Contain — optional network isolation after documentation is saved locally.
  4. Escalate check — compare findings to the criteria below.
  5. Block / quarantine / harden — only if DIY is still appropriate.

When to escalate

Consider professional help if any of these apply. Full flowchart: SVG version (exact labels).

  • Data theft, financial loss, or ransomware? → Escalate immediately. Prepare evidence package.
  • Confirmed persistence + ongoing network beaconing? → Escalate (strong recommendation).
  • Personally targeted or feel unsafe? → Escalate.
  • Multiple future-timestamp files + suspicious scripts + odd network activity? → Consider escalating, especially if activity continues.
  • Still unsure after reviewing everything? → Escalate (better safe than sorry).

Platform safety — block vs mute (X)

Defensive actions on your accounts only. Screenshot and log before block or report — moderation may remove posts (~85%). Full platform guide.

Critical sequence: Screenshot + log → mute (if you want quiet) → report (if ToS violated) → block (if engagement must stop).

Breach check (Have I Been Pwned)

k-Anonymity — how the prefix search works

HIBP compares password hashes without sending your full password. Only the first 5 characters of the SHA-1 hash leave your browser (~85%). That narrows results to a bucket of possible matches — you compare the remainder locally.

Prefix sent · remainder compared offline on your device

  • Rotate passwords on a clean device after documentation.
  • Never paste live passwords into chat or public repos.
